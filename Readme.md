# Mintlify Homepage Clone

## Fonts Used
- **Inter**
  - Loaded via Google Fonts
  - Weights used: 100, 200, 300, 400, 500, 600, 700
  - Applied globally for clean, modern typography

## Colors Used

### Backgrounds
- `#FFFFFF` – Primary page background
- Light SVG background from Mintlify hero section

### Text
- `#000000` – Primary headings and body text
- `#9A9A9A` – Muted text, secondary labels, footer text
- `#717070` – Scrollbar thumb

### Brand / Accent
- `green` – Accent color used in icons, highlights, status indicators
- `currentColor` – SVGs inherit parent text color

### Buttons
- Primary CTA: white text on dark background
- Secondary CTA: transparent background with border

### Borders & Dividers
- `#CFCFCF` – Dashed and solid circular borders
- Soft neutral borders for cards and footer sections

## Icons & Graphics
- SVG icons used for scalability and theme consistency
- Inline SVGs allow color control via CSS
- Hero and illustration images loaded via external SVG and PNG assets

## Layout & Styling Notes
- CSS variables defined in `:root` for font management
- `box-sizing: border-box` applied globally
- Flexbox used for navigation, grids, and alignment
- Absolute positioning used for hero background SVG layering
- Responsive width handling using percentage-based sizing

## Scrollbar Styling
- Custom scrollbar thumb color
- Transparent track for minimal UI distraction
